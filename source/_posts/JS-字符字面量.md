# 字符字面量 #

| 字面量 |                含义                 |
| :----: | :---------------------------------: |
|   \n   |                换行                 |
|   \t   |                制表                 |
|   \b   |                退格                 |
|   \r   |                回车                 |
|   \f   |                换页                 |
|   \\   |             反斜杠（\）             |
|   \'   |             单引号（'）             |
|   \"   |             双引号（"）             |
|   \`   |             反引号（`）             |
|  \xnn  |     以十六进制编码nn表示的字符      |
| \unnnn | 以十六进制编码nnnn表示的Unicode字符 |
