GIT

GIT配置

  • window安装后点击鼠标右键多了两个右键菜单（Git GUI Here 与 Git Bash Here）表示安装成功

  • Mac系统，打开终端，输入 git –version 回车，看到Git版本号表示安装成功

必要配置

Git要求，必须配置一个用户名和一个邮箱

当我们使用Git记录代码的版本的时候，Git需要知道你是谁？也就是提交代码的人是谁？
用户名与邮箱，理论上随意配置，但最好配置成和你的 码云、Github相同的用户名与邮箱；

  1. windows用户，在任何文件夹空白处，鼠标右键，打开 “Git Bash Here”; Mac用户打开终端窗口
  2. 在出现的终端窗口中，输入 git config --global user.name "xxx" ，配置用户名
  3. 在出现的终端窗口中，输入 git config --global user.email "xxx" ，配置邮箱

假设我们开发一个项目，首先进入到项目目录中
文件夹空白处，鼠标右键，打开 “Git Bash Here” 窗口
执行 git init 命令，完成仓库的初始化
